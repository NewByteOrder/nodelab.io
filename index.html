What is NodeLab?

NodeLab is data processing platform powered by Luna programming language.
It redefines how you think about data processing. It enables you to think and code visually bridging the communication gap between technical and non-technical staff.

Rocket science technology

Luna, the world's first high performance, hybrid visual-textual programming language, is the NodeLab backbone. It is as fast as C++, as easy as Python and provides bindings to the tools you already know, including Python and R.

Strong and extensible base

We provide you with a broad set of ready-to-use algorithms including statistics, machine learning and linear algebra. If something isn't there you are free to implement it in Luna.

Take it anywhere

Our platform runs just in your web browser while running computation in the cloud. We also provide native apps for Windows, Mac and Linux.

Powerful team

NodeLab is built by the team of over 20 brilliant engineers with strong background in data processing.
